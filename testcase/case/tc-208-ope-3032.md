# TC-208: Antigravity agy runtime local run support（OPE-3032）

## 关联信息

- **OPE 编号**: OPE-3032
- **Gitee PR**: !408

- **特性摘要**: `multica run` 支持 Antigravity (`agy`) runtime，并将 transcript 同步为 issue comment 时过滤原始 tool output。

## 涉及源文件

- `server/cmd/multica/cmd_run_agy.go`
- `server/cmd/multica/cmd_run_provider.go`
- `server/cmd/multica/cmd_run_test.go`

## 验证要点

1. 本地存在可执行 `agy` 或测试替身时，`multica run --provider agy` 能启动 Antigravity runtime。
2. 运行过程中的 transcript tracker 能记录消息并同步最终 issue comment。
3. 最终 issue comment 不包含原始 tool output，只保留用户可读结果。
4. provider 选择、错误提示、退出码和已有 Claude/Codex local run provider 行为保持兼容。
5. 单元测试和集成测试覆盖 tracker 启动、消息过滤和无原始输出泄漏的回归场景。
