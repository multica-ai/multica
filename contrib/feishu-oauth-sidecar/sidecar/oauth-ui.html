遇到问题？ 诊断 / 强制重置 ▾

什么时候用强制重置？

  • 授权扫码后页面卡住、长时间不切到"成功"
  • 已绑定但 multica 操作仍报"飞书授权失效"
  • 想换飞书账号（清旧绑定后重新扫码）
  • 在另一台机器扫码了，本机没同步

强制重置会做什么？

  • 清除当前 multica 用户的飞书 token 凭据
  • 清除 mapping (本机数据库的飞书身份记录)
  • 保留你的其他用户数据 (只删 .lark-cli / .hermes/auth.json / .config/lark)
  • 之后请点上方"开始飞书授权"重走扫码流程
