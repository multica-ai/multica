Workspace mobile navigation preview

这个示意稿只用来确认移动端工作区的层级关系。方向更接近 Jira： 顶部只保留工具栏，一级导航收进侧滑菜单，当前页面自己的菜单放在内容上方。

重点确认三件事：顶部工具栏是否足够轻、一级导航是否应该收进侧滑菜单、 以及双栏页面在手机上是否统一改成“列表进入详情再返回”的单列结构。

1. Issues main view

顶部只放菜单入口、工作区名和新建入口。当前页面的范围切换和筛选操作放在内容上方， 不把整套一级导航铺在顶部。
