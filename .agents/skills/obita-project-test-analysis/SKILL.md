## 模板资源

- 生成“业务泳道图与测试关注点”章节时，固定加载 `assets/test-analysis-swimlane-section-template.html`。
- 该模板是单章节 HTML，只包含泳道图及其配套测试关注点表，不包含完整测试分析文档的其他章节。
- HTML 支持内联 SVG；泳道图可直接用 `<svg>`、`<rect>`、`<text>`、`<path>` 和 `<marker>` 绘制，不依赖 Mermaid 或外部图片资源。
- 泳道图表达规则：纵向泳道固定表达角色或责任方，横向不设置角色或阶段表头，流程节点按业务发生顺序自然向右推进。
- 使用模板时替换 `{{...}}` 占位符，并按真实业务保留、删除或扩展泳道、步骤和测试关注点。
