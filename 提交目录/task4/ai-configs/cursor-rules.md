# Cursor Rules

## Repository Rules

- Read `CLAUDE.md` before editing any shared code.
- Keep state rules intact: server state in React Query, client state in Zustand.
- Keep package boundaries clean.
- Prefer existing patterns over new abstractions.

## Daily Workflow

1. Inspect the smallest relevant files first.
2. Write or update tests before changing behavior.
3. Verify the narrowest useful command.
4. Expand only when the first pass is stable.
