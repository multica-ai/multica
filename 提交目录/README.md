# 提交目录

这份目录按照题目要求组织，包含四个任务的交付物：

- `task1/`：本地搭建日志与故障场景推演
- `task2/`：任务编排引擎设计文档，以及可运行的核心实现代码和测试
- `task3/`：修复分析报告与 PR 链接占位

- `fork-repo-link.txt`：fork 仓库地址占位

- `task2/src/` 里放的是一个独立、纯内存的 DAG 编排引擎实现，重点覆盖依赖构建、环检测、调度、并发限制和失败策略。
